# 冲突记录模板

> 当两条正式记忆相互矛盾时，在此目录记录冲突并标记待裁决。
> 记忆条目本身的 `status: conflict` 也用于标记单条记忆处于冲突态。

复制本模板改名后填写：

---

## Conflict: <冲突主题>

- Found at: YYYY-MM-DD
- Found by: <name>
- Source A: `<记忆 id 或文件路径>`
- Source B: `<记忆 id 或文件路径>`

### 冲突描述

- A 记忆说：……
- B 记忆说：……

### 建议处理

<需要确认哪个为准 / 是否都已过期 / 是否需要新决策替代>

### Status

pending_review
